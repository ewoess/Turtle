Commands

FD n / BK n — forward/back n px

RT a / LT a — turn right/left a degrees

PU / PD — pen up/down

HOME — go to (0,0), heading 0 (up)

CS — clear screen

SETPOS x y — move to x,y (center = 0,0; +y up)

SETHEADING d — 0 up, 90 right (Logo‑style)

PENCOLOR r g b — set pen (0‑255)

PENSIZE n — 1..12

RAINBOW ON/OFF — auto‑cycle color each segment

HUESTEP n — hue step in degrees (default 5)

REPEAT n [ ... ] — supports nesting
